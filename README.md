# Server-Side Rendering using Node.js, Express, and EJS

This project demonstrates a basic web application that implements **server-side rendering** using **Node.js**, **Express.js**, and **EJS**. It includes an HTML form for user input, server-side processing of form data, and dynamic rendering of HTML pages. This project is developed for academic and learning purposes.
